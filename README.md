# پروژه‌ها

این مخزن با هدف بایگانی پروژه‌های اعضای گروه کاربران گنو/لینوکس تهران ایجاد شده است.  
هدف این مجموعه، دیده‌شدن، حمایت و همکاری در پروژه‌هاست و انتظار می‌رود تأثیری مثبت بر فعالیت‌های اعضا در دنیای اوپن سورس داشته باشد.
